Project Descripition

Login Page :

The Login page ensures secure authentication for users accessing personalized features of the application. It validates user credentials through Spring Boot authentication APIs and manages user sessions efficiently. Upon successful login, users gain access to features such as wishlist management, cart operations, and order history, creating a personalized shopping experience.

Product Details Page

The Product Details page provides complete information about a selected product to help users make informed purchasing decisions. It displays the product image, name, price, category, deal tags, and detailed description. Users can select quantities and seamlessly add items to their cart or wishlist. The page is designed for clarity and smooth navigation, allowing users to return to the product listing effortlessly.

Wishlist Page The Wishlist page allows authenticated users to save products they are interested in for future reference. All wishlisted items are displayed in an organized layout, with options to remove products or directly add them to the cart. Wishlist data is stored persistently in the PostgreSQL database, ensuring saved items remain available across sessions.

Orders Page: The Orders page displays a complete history of purchases made by the user. Each order includes relevant details such as order date, total amount, and purchased products. The page is powered by secure backend order management APIs and is accessible only to authenticated users, ensuring data privacy and reliability. Cart Page: The Cart page enables users to review and manage selected products before placing an order. It displays product details along with quantity and pricing information, allowing users to update quantities or remove items as needed. The total price is calculated dynamically, providing users with a clear order summary before proceeding to checkout.Orders Page:
